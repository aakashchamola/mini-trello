# Mini Trello

A minimal Trello-like app using Node.js and Express.

## Setup

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the server:
   ```sh
   npm start
   ```

## Endpoints
- `/` - Welcome message
- `/boards` - Sample boards endpoint

## Development
Run with auto-reload:
```sh
npm run dev
```
